# Email-Spam-Detection
📧 Email Spam Detection A Machine Learning-powered classifier using NLP to filter "Spam" from "Ham" messages.  Core: Built with Python &amp; Scikit-Learn.  Tech: Uses TF-IDF vectorization and Naive Bayes for 98%+ accuracy.  Workflow: Data cleaning, text preprocessing, and real-time prediction.  Goal: High-precision filtering to keep inboxes clean.
